# Meu Projeto Flask

Este é um site básico desenvolvido com Flask, HTML, CSS e SQLite para aprendizado de desenvolvimento web.

## Funcionalidades
- Página inicial.
- Página de contato com formulário.
- Banco de dados SQLite para armazenar mensagens de contato.
